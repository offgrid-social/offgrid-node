# offgrid-node

Reference node implementation for the offgrid network.

Nodes handle storage, validation, federation, and peer communication.

## Scope
- Node runtime
- Data storage
- Federation logic
- Peer-to-peer communication

## Notes
This implementation can be forked or replaced.
Compatibility is defined by `offgrid-core`, not by this codebase.
